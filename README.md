# Customer Analytics & Campaign Performance Dashboard (Power BI)

## Overview
This project demonstrates my ability to transform raw datasets into structured, analysis-ready data models and deliver actionable insights through interactive dashboards. It simulates a real-world data operations workflow, including data ingestion, cleaning, transformation, and reporting.

## Objectives
- Analyze customer conversion behavior for a marketing campaign  
- Identify high-performing customer segments  
- Evaluate the effectiveness of outreach strategies  
- Provide data-driven recommendations for campaign optimization  

## Data Pipeline & Transformation
- Ingested raw dataset from external source (CSV)
- Performed data cleaning (handling missing values, standardizing categories)
- Transformed and modeled data using Power Query
- Created calculated columns and measures using DAX
- Validated data for consistency and accuracy before analysis

## Tools & Technologies
- Power BI (Data Modeling & Visualization)
- Power Query (ETL / Data Transformation)
- DAX (Measures & KPIs)
- Excel (Data preparation)

## Key Insights
- Identified customer segments with the highest conversion rates  
- Highlighted inefficiencies in outreach channels  
- Revealed patterns in customer demographics and campaign success  

## Business Impact
- Demonstrates how structured data and analytics can improve campaign ROI  
- Enables stakeholders to make informed, data-driven decisions  
- Reduces reliance on manual reporting through automated dashboards  

## Deliverables
- Interactive Power BI dashboard (.pbix)
